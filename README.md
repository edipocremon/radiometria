<p><b>Radiometria</b></p>
<p><br></p>
<p>Nesta página é disponibilizado os arquivos mensurados no experimento da radiometria da disciplina Sensoriamento Remoto-1 do Instituto Federal de Goiás (Campus Goiânia).</p>

</ul>
